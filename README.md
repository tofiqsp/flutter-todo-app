# Todo App

This is a Flutter application that helps you manage your tasks and stay organized.

## Installation

To install the app, follow these steps:

1. Clone the repository to your local machine.
2. Open the project in your preferred IDE (such as Android Studio or VS Code).
3. Run `flutter pub get` to install the required dependencies.
4. Run `flutter run` to start the app.

## Features

The Todo App has the following features:

- Create, edit, and delete tasks.
- Set due dates and reminders for tasks.
- Mark tasks as completed.
- Organize tasks into lists.

## Technologies Used

The Todo App is built using the following technologies:

- Flutter
- Dart
